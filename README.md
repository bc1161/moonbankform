# HSBC-ROI-Project
A training project for the 2023 graduate program. A web application to allow clients to request a bank account to be created


TO run the Server

### `cd Server`

### `python3 app.py`


To run the Client

### `cd Client`

first run to install node modules
### `npm install`

Then run
### `npm start`
 to run te code in development mode
